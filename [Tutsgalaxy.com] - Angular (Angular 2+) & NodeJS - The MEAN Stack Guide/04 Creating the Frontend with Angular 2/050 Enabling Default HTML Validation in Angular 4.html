In the following video, we'll add validation to our form, here's an important note about that lecture:

If you're using Angular 4, the default HTML validation will be disabled by Angular. You can easily enable it again though, simply add thengNativeValidate directive to the <form> element.

Like this:

<form ngNativeValidate>
    ... Your form code ...
</form>

With this, HTML5validation should be working again.
